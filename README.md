# Elderly Companion Matching System

## Overview

The Elderly Companion Matching System is a blockchain-based intergenerational companionship program designed to safely connect elderly individuals with volunteer companions while incorporating robust safety features and non-intrusive wellness monitoring.

## Project Description

This decentralized platform facilitates meaningful connections between elderly community members and volunteer companions through secure, blockchain-verified matching processes. The system prioritizes safety, privacy, and genuine human connection while leveraging smart contract technology to ensure transparency and accountability.

## Key Features

### 🤝 Safe Companion Matching
- Verified volunteer screening process
- Compatibility-based matching algorithms  
- Background check integration
- Community reputation system
- Emergency contact management

### 💚 Wellness Monitoring
- Non-intrusive check-in protocols
- Automated wellness alerts
- Family notification systems  
- Health trend tracking
- Emergency response coordination

## Smart Contracts

### 1. Companion Matching Contract (`companion-matching.clar`)
- **Purpose**: Safe matching of elderly individuals with companion volunteers
- **Features**:
  - Volunteer registration and verification
  - Elderly participant onboarding
  - Matching algorithm implementation
  - Safety protocol enforcement
  - Match history tracking

### 2. Wellness Monitoring Contract (`wellness-monitoring.clar`)
- **Purpose**: Non-intrusive wellness monitoring and check-ins
- **Features**:
  - Regular wellness check protocols
  - Alert system for missed check-ins
  - Health trend analysis
  - Emergency contact notifications
  - Privacy-preserving data handling

## Getting Started

### Prerequisites
- Clarinet CLI installed
- Node.js and npm
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/olamidefaruk041/elderly-companion-matching.git
cd elderly-companion-matching
```

2. Install dependencies:
```bash
npm install
```

3. Run contract checks:
```bash
clarinet check
```

4. Run tests:
```bash
clarinet test
```
